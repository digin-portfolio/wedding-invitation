## ✨ Features

- 💌 Animated envelope opening with wax seal effect
- ⏳ Live countdown timer to the wedding date
- 📅 Event timeline with schedule
- 📍 Venue location with embedded Google Maps
- 🗓️ Add to Calendar button
- 💬 Congratulations message section
- 🌸 Floating hearts & sparkle animations
- 🎵 Background music on open
- 📱 Fully responsive — mobile friendly

## 📁 File Structure
```
├── index.html      # Main invitation page
├── style.css       # All styles and animations
├── script.js       # Countdown, envelope, interactions
└── A_Thousand_Years.mp3  # Background music (add manually)
```

---

## ⚙️ How to Customize

1. Open `index.html`
2. Replace couple names, date, and venue
3. Update Google Maps embed link
4. Replace `A_Thousand_Years.mp3` with your preferred song
5. Deploy on GitHub Pages or any static host

---

## 🚀 Deploy on GitHub Pages

1. Push files to a GitHub repo
2. Go to **Settings → Pages**
3. Select **main branch → root**
4. Your invitation is live at `yourusername.github.io/repo-name`
